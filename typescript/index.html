Introduction to TypeScript

TypeScript is a programming language developed and maintained by Microsoft. It is a strict syntactical superset of JavaScript and adds optional static typing to the language.

Basic TypeScript Syntax


function greeter(person: string) {
    return "Hello, " + person;
}

let user = "User";

document.body.textContent = greeter(user);

TypeScript Concepts

  • Types: TypeScript adds types to JavaScript, like string, number, boolean, etc.
  • Interfaces: A way to define contracts for what an object should look like.
  • Classes: TypeScript supports object-oriented programming with classes.
  • Compilation: TypeScript code needs to be compiled into JavaScript to run in a browser.

Compiling TypeScript

To run TypeScript code in a browser, you need to compile it to JavaScript. First, you need to have Node.js and npm (Node Package Manager) installed. Then, you can install TypeScript globally using npm:

npm install -g typescript

Save your TypeScript code in a file with a .ts extension, for example greeter.ts. Then, compile it to JavaScript using the command:

tsc greeter.ts

This will create a greeter.js file, which you can then include in your HTML file.
