Introduction to Python

Python is an interpreted, high-level and general-purpose programming language. Python's design philosophy emphasizes code readability with its notable use of significant whitespace.

Python Concepts

  • Variables and Data Types: Python is dynamically typed. Common types are integers, floats, strings, lists, dictionaries.
  • Functions: Defined with the def keyword.
  • Control Flow: Use if, for, and while for conditional logic and loops.
  • Modules and Packages: Import other Python files to use their functions and variables.

Running Python Code

To run Python code, you need to have Python installed on your computer. You can download it from python.org.

Save the code in a file with a .py extension, for example hello.py. Then open a terminal or command prompt and run the file using the command:

python hello.py
